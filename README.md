# holbertonschool-web_front_end

## HTML Advanced

This project focuses on learning the fundamentals of HTML and how to structure a webpage using semantic HTML5 elements.

The goal of this project is to understand how to correctly build the structure of a webpage without using CSS or styling. The pages created in this project may look visually simple, but the objective is to practice writing clean, valid, and well-structured HTML code.

## Learning Objectives

By the end of this project, we should be able to:

- Understand the basic structure of an HTML5 document
- Use semantic HTML elements to organize webpage content
- Apply proper heading hierarchy
- Create lists and tables
- Embed images, videos, and audio in a webpage
- Embed external content
- Write HTML that validates with the W3C Validator

## Requirements

- All HTML code must be W3C compliant
- A README.md file must exist at the root of the repository
- All files must follow proper HTML syntax and structure
- The project will use **Techium** as the example company for the webpages
